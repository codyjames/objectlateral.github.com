AngularJS

Foundations Course

A three day, hands-on, intense introductory course on buidling modern web application with AngularJS.

During the course, students will advance through a series of lectures, challenges, and labs which gradually introduce AngularJS topics and build a solid foundation for building AngularJS applications.

Topics

  • setup & tools
  • controllers & scopes
  • data bindings
  • filters
  • built-in & custom directives
  • factories, services, & providers
  • routing & templating
  • testing
  • dependency injection
  • $http & promises
  • ngResource
  • forms & form validations
  • animations
  • events, $broadcast, $emit, & $watch

Themes

In addition to these topics, the course features two themes.

Testing

Testable applications are maintainable applications. While this course has times dedicated specifically to testing, the theme of testing is woven throughout the course's fabric.

Building

Wisdom is knowledge applied. During the course, students build a real, production-ready application from the ground up. See the labs section for details.

Lectures

15-30 minutes

Each session starts with a brief lecture on one or more AngularJS topics. Lectures are composed of visual aids, analogies, code samples, and occasional live coding to keep things interesting.

Lectures come in three flavors:

1) Intro

An intro's sole purpose is to introduce a topic and provide context for the subsequent hands-on sessions.

2) Deep Dive

Certain topics are worthy of further exploration than can fit in to an Intro lecture. Often times these lectures focus on the philosophy behind how AngularJS components work.

3) Review

The key to learning is repetition. Review times are dedicated to the previously focused on topics and always come at the end of a session.

Challenges

30-45 minutes

Challenges are standalone, achievable programming tasks that utilize the previously lectured topic. Students work together (or alone) to complete the tasks in the time allotted.

Challenges also give students the opportunity to work alongside the instructor, ask questions, and get feedback on their solutions.

Sample Challenges

  1. Use a service to share data between two controllers
  2. Given a failing test suite, write code to make tests pass
  3. Given an implemented feature, add unit tests to it
  4. Write a custom directive from the provided controller code
  5. Wrap a 3rd party API call with a service

Labs

30-45 minutes

Labs account for a large portion of the course. During this time, students build a real-world, production ready AngularJS application from the ground up.

Each lab time will begin with a set of user stories to be implemented in the time allotted. The stories progress in scope and required knowledge alongside the lecture material.

Unlike the Challenges, the lab work will be ongoing and students can use the time however they think is best. Students who excel can move on to more advanced user stories and those who need extra focus on a particular piece of AngularJS can do just that.

The goal for labs is for each student to have built an AngularJS application that utilizes all topics covered which they can use for future reference and learning.

Sample Lab Applications

  1. An outline creation tool
  2. A multiplayer Connect Four game
  3. A simple finance analysis tool
